# 股市技术分析学习路线（三）：技术指标——用数学工具量化趋势与买卖时机

> 阶段一学会了"认字"（K线），阶段二学会了"认句"（图表形态），阶段三要学的是"量尺"——用数学公式把直觉判断变成可量化的指标。当你在图表上画出均线、MACD、RSI的那一刻，你就不再只凭"感觉"做判断，而是有了客观的参考系。这个阶段大约需要3-4周，学完后你将拥有系统化的买卖信号判断能力。

---

## 本阶段学习目标

- 理解每个核心指标的计算逻辑（不需要手算，但必须知道公式含义）
- 看到指标信号，能在5秒内判断其含义和可靠程度
- 掌握指标的"金叉/死叉""超买/超卖""背离"三大核心信号
- 理解每个指标的适用场景和盲区，不再盲目迷信指标
- 能组合2-3个指标形成自己的"信号确认体系"

---

## 技术指标总览

技术指标按功能分为四大类：

| 分类 | 功能 | 代表指标 | 回答的问题 |
|------|------|----------|------------|
| **趋势型** | 判断趋势方向和强度 | MA、EMA、MACD | "趋势还在吗？方向是什么？" |
| **动量型** | 衡量价格变化的速度 | RSI、KDJ、威廉指标 | "涨/跌的速度是加快还是减慢？" |
| **波动率型** | 衡量价格波动的剧烈程度 | 布林带、ATR | "波动在放大还是收缩？" |
| **成交量型** | 确认价格运动的可靠性 | OBV、成交量均线 | "这个涨/跌有没有资金支撑？" |

**核心原则**：
1. **指标是辅助，不是圣杯**——任何指标都有失效的时候
2. **指标滞后于价格**——所有指标都是用历史数据算出来的，必然滞后
3. **多指标共振才可靠**——单一指标信号可靠性低，2-3个指标同时发出信号才值得操作
4. **先定趋势，再用指标**——趋势型指标优先，在趋势明确的前提下使用动量型指标找买卖点

**多头排列**：所有均线从上到下按短→长排列，像一队整齐上行的士兵 → **强势上涨趋势**
**空头排列**：所有均线从下到上按短→长排列，像一队整齐下行的败兵 → **强势下跌趋势**

### 1.5 常用均线参数

| 参数 | 用途 | 特点 |
|------|------|------|
| **MA5** | 超短期趋势 | 跟随价格极紧，频繁发出信号 |
| **MA10** | 短期趋势 | 短线交易者常用 |
| **MA20** | 月线趋势 | 中线买卖点参考（布林带中轨） |
| **MA60** | 季线趋势 | 中长线趋势判断（牛市/熊市分界） |
| **MA120** | 半年线 | 长线趋势，主力成本参考 |
| **MA250** | 年线 | 牛熊分界线，最重要的长期均线 |

### 1.6 均线的实战要点

- **MA20以上只做多，MA20以下只做空**——这是最简单的趋势跟踪规则
- **金叉/死叉的可靠性与周期正相关**——周线金叉远比5分钟金叉可靠
- **均线的支撑/阻力作用**——价格回踩MA20/MA60常有支撑
- **均线斜率**比位置更重要——平缓的均线意味着趋势不明，陡峭的均线意味着趋势强劲

### 2.2 MACD的计算逻辑

```
DIF（快线）= EMA(12) - EMA(26)
DEA（慢线）= EMA(DIF, 9)
MACD柱  = (DIF - DEA) × 2
```

**通俗理解**：
- DIF = 短期均线和长期均线的"距离"→ 趋势的"加速度"
- DEA = DIF的平滑版 → 趋势的"速度"
- MACD柱 = DIF和DEA的差 → "加速度的变化"

### 2.5 MACD的实战用法

| 信号 | 含义 | 可靠性 | 操作建议 |
|------|------|--------|----------|
| 零轴上方金叉 | 强势中的买入 | ⭐⭐⭐⭐⭐ | 积极买入 |
| 零轴下方金叉 | 弱势中的反弹 | ⭐⭐⭐ | 小仓试探 |
| 零轴上方死叉 | 强势回调 | ⭐⭐⭐ | 减仓，不恐慌 |
| 零轴下方死叉 | 弱势中的卖出 | ⭐⭐⭐⭐⭐ | 果断卖出 |
| 顶背离 | 上涨动力衰竭 | ⭐⭐⭐⭐ | 逐步减仓 |
| 底背离 | 下跌动力衰竭 | ⭐⭐⭐⭐ | 逐步建仓 |

**⚠️ MACD的局限**：
- 严重滞后——等到MACD死叉时，价格往往已经跌了一大段
- 震荡市中频繁发出假信号——金叉死叉反复出现
- **解决办法**：结合趋势判断（均线方向）+ 更大周期确认（周线MACD）

## 三、RSI（相对强弱指标）⭐⭐⭐⭐⭐

**最常用的动量型指标，衡量"涨过头"还是"跌过头"。**

### 3.1 RSI的计算与含义

```
RSI = 100 - 100/(1 + RS)
RS = N日内平均涨幅 / N日内平均跌幅
```

**通俗理解**：RSI衡量的是过去N天里，涨的日子力度大还是跌的日子力度大。

### 3.3 RSI背离

RSI背离与MACD背离逻辑相同，但RSI背离往往**更早出现**：

| 背离类型 | 表现 | 含义 | 可靠性 |
|----------|------|------|--------|
| **顶背离** | 价格创新高，RSI没创新高 | 上涨动力衰竭 | ⭐⭐⭐⭐ |
| **底背离** | 价格创新低，RSI没创新低 | 下跌动力衰竭 | ⭐⭐⭐⭐⭐ |

**为什么底背离比顶背离更可靠？** 因为下跌到极端时，恐慌情绪释放得更彻底，反转更确定；而上涨中贪婪可以持续很久。

## 四、KDJ（随机指标）⭐⭐⭐⭐

**捕捉短期超买超卖的利器，中国A股使用最广泛的动量指标之一。**

### 4.1 KDJ的构成

```
RSV = (收盘价 - N日最低价) / (N日最高价 - N日最低价) × 100
K = 2/3 × 前日K + 1/3 × RSV
D = 2/3 × 前日D + 1/3 × K
J = 3K - 2D
```

## 六、成交量指标

### 6.1 OBV（能量潮指标）⭐⭐⭐⭐

**"量在价先"——OBV把成交量变成一条趋势线。**

```
如果今天收盘>昨天收盘：OBV = 昨日OBV + 今日成交量
如果今天收盘<昨天收盘：OBV = 昨日OBV - 今日成交量
如果今天收盘=昨天收盘：OBV = 昨日OBV
```

**OBV的核心用法**：
- OBV与价格同涨同跌 → 趋势健康
- OBV涨而价格不涨 → 资金在暗中买入，后续可能上涨
- OBV跌而价格不跌 → 资金在暗中出逃，后续可能下跌
- OBV创新高而价格未创新高 → 顶背离，看跌

```
TR = max(当日最高-当日最低, |当日最高-昨收|, |当日最低-昨收|)
ATR = TR的N日移动平均（通常N=14）
```

## 十一、阶段三实战练习

### 11.1 每日必做

1. **选择3只股票**，分别叠加MA5/20/60、MACD、RSI指标
2. **判断当前趋势**：均线排列是多头还是空头？
3. **检查MACD**：DIF在零轴上方还是下方？有没有背离？
4. **检查RSI**：处于什么区间？有没有背离？
5. **综合判断**：三个指标的信号是否一致？

### 11.2 指标信号识别训练

| 场景 | 你的判断 |
|------|----------|
| MA5上穿MA20 + MACD零轴上方金叉 | ? |
| 价格创新高 + RSI顶背离 | ? |
| 布林带收窄 + MACD零轴下方 | ? |
| KDJ超卖区金叉 + OBV持续上升 | ? |
| MA60上方 + RSI从30回升至50 + MACD绿柱缩短 | ? |

### 11.3 常见错误与纠正

| 错误 | 纠正 |
|------|------|
| RSI到70就卖 | 牛市RSI可长期>70，等RSI背离再卖 |
| MACD金叉就买 | 先看均线方向，空头排列中的金叉不可靠 |
| 同时用5个动量指标 | 同类指标只选一个，不同类型组合 |
| 只看日线指标 | 至少看日+周两个周期 |
| 迷信参数优化 | 默认参数经过数十年验证，比自编参数可靠 |
| 不设止损 | 用ATR设定科学止损位 |
| 指标信号与价格矛盾时信指标 | 永远以价格为准，指标只是辅助 |

## 十二、阶段三自我测试

完成以下测试后，就可以进入阶段四（量价关系深度解读）：

### 基础题（必须全对）
1. MACD的DIF线是怎么算出来的？它在零轴上方意味着什么？
2. RSI的超买超卖标准值是多少？为什么牛市中RSI可以长期超买？
3. 布林带收窄意味着什么？收窄后一定会涨吗？
4. MA5和MA20的金叉在什么情况下最可靠？
5. 为什么同类指标（如RSI和KDJ）不应该同时使用？

### 进阶题（至少对4/5）
1. MACD顶背离和RSI顶背离哪个更早出现？为什么？
2. ATR是0.8元的股票和ATR是3元的股票，止损位应该怎么设？
3. 布林带开口方向与趋势方向一定一致吗？举例说明。
4. 为什么"均线之上只做多"是一个简单但有效的规则？
5. 如果MACD金叉但RSI在超买区，你该怎么操作？

### 实战题
打开任意A股日K线图，完成以下操作：
1. 叠加MA5/20/60，判断当前均线排列
2. 打开MACD，判断DIF位置、金叉/死叉状态、有无背离
3. 打开RSI，判断当前区间和方向
4. 综合三个指标，给出当前的操作建议（买入/持有/卖出/观望）
5. 用ATR设定止损位
6. 一周后回来验证你的判断

---

**答案要点**：

基础题：
1. DIF = EMA(12) - EMA(26)；零轴上方意味着短期均线在长期均线之上，多头市场
2. 超买>70，超卖<30；牛市中买盘强劲，RSI可以长期停留在高位
3. 收窄意味着波动率收缩，多空蓄力即将变盘；不一定涨，也可能向下突破
4. 在多头排列中、零轴上方出现的金叉最可靠
5. 同类指标数学原理相似，信号高度相关，叠加使用不增加信息量，反而增加假信号

进阶题：
1. RSI背离通常更早出现，因为RSI只考虑单周期涨跌幅，MACD有多重平滑所以更滞后
2. 止损位 = 买入价 - 2×ATR。ATR 0.8元的止损1.6元，ATR 3元的止损6元
3. 不一定。布林带开口方向反映波动率扩张，短期可能反向刺穿后才开始真正趋势
4. 因为均线之上说明短期成本>长期成本，多头占优，逆势做空胜率低
5. 谨慎操作。金叉看涨但RSI超买说明短期涨幅过大，可能先回调再上涨。策略：等回调到MA20再买入
